Sobre mí

Soy Felipe Levi, desarrollador Full Stack Jr con formación en Ingeniería Civil Electrónica y experiencia en automatización industrial e I+D. En mis proyectos desarrollo un enfoque metódico y orientado a resultados. Manejo Python, Django, HTML, CSS, JavaScript y bases de datos SQL. Mini-X (Red Social)

Mini-X es una aplicación web inspirada en redes sociales modernas, donde los usuarios pueden crear publicaciones, dar “me gusta” y dejar comentarios. El proyecto se centra en la experiencia de usuario y en la creación de una interfaz limpia y funcional. Está construido para practicar CRUD completos, relaciones entre modelos y diseño responsivo.

Gestor de Productos

El Gestor de Productos es una aplicación web creada con Django para administrar un catálogo de productos de forma sencilla. Permite registrar usuarios, crear y editar productos, y organizarlos mediante categorías y etiquetas. Incluye funciones de filtrado por caterorías y etiquetas, y una interfaz clara enfocada en la gestión básica de inventarios.
